## 📖 Sobre o Repositório

Este repositório serve como um **arquivo central (armazém)** para todos os códigos e soluções que desenvolvo para os problemas da plataforma [Beecrowd](https://www.beecrowd.com.br/) (antigo URI Online Judge). 

O objetivo principal de manter essas soluções é:
- Acompanhar minha evolução em lógica de programação e otimização de algoritmos.
- Servir como material de consulta rápida para estruturas de dados, métodos de ordenação e problemas matemáticos complexos.
- Praticar a implementação eficiente em diferentes linguagens, com foco em redução de tempo de execução e uso de memória.

## 🛠️ Tecnologias Utilizadas
A maior parte das soluções foca no uso de linguagens eficientes para maratonas (como Java, C++ ou Python), aproveitando as capacidades de cada uma para resolver desafios que vão de **Iniciante** a **Grafos** e **Geometria Computacional**.

---

## 🚀 Como Explorar

Sinta-se livre para explorar os arquivos de código-fonte. Cada arquivo geralmente possui o nome/número do problema no Beecrowd para facilitar a identificação.

> **Importante:** Se você também está aprendendo ou resolvendo problemas no Beecrowd, recomendo tentar chegar à solução sozinho antes de consultar o código. A programação competitiva é sobre desenvolver a *sua* capacidade analítica!
